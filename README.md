
#  Document QnA with LLaVA  

This project extracts text from PDf and doc.x files, breaks it into chunks, and then uses the l LLaVA model (via Ollama) to answer user questions based on the document content.  
It works completely offline once models are downloaded.


✔ Extract text from PDF  
✔ Extract text from DOCX  
✔ Smart text chunking  
✔ Finds best related chunk to the query  
✔ Uses LLaVA model locally  
✔ No API, No Internet Required
